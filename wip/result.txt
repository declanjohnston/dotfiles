== Table Explanations ==

- message: The core message table. One row per message.
- handle: Maps phone numbers / Apple IDs to internal IDs.
- chat: Represents a thread (group chat or one-on-one).
- chat_message_join: Many-to-many join between chats and messages.

JOIN logic:
  message.handle_id → handle.ROWID
  chat_message_join.message_id → message.ROWID
  chat_message_join.chat_id → chat.ROWID
